# GitHub Pages Blog Starter

This repo is ready for GitHub Pages.

- Uses the **minima** theme (built-in).
- Posts live in the `_posts/` folder. File name format: `YYYY-MM-DD-your-title.md`.
- Home page lists your posts automatically.

## Quick edits
- Change site title/description in `_config.yml`.
- Edit `about.md` for your bio.
- Add new posts in `_posts/`.
